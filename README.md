### Installation

This app requires [Node.js], [express.js], [mongoDB], [mongoose.js], [passport.js]

Install the dependencies and devDependencies and start the server.

```sh
$ cd express
$ npm install
$ npm start
```

And then you will navigated to the browser at localhost.

```sh
localhost:3000
```
